üéß Example Usage

JavaScript Fetch:

fetch('https://iamaanahmad.github.io/NasheedCollection/audio_catalog.json')
  .then(response => response.json())
  .then(data => {
    console.log('Total files:', data.metadata.total_files);
    console.log('First file:', data.files[0]);
  });

HTML Audio Player:

<audio controls>
  <source src="https://cdn.jsdelivr.net/gh/iamaanahmad/NasheedCollection@master/filename.mp3" type="audio/mpeg">
</audio>

üìö Catalog Structure

Each file in the catalog includes:

  • id: Unique identifier
  • filename: Original filename
  • title: Clean title
  • artist: Artist name
  • urls: Multiple access URLs (raw, cdn, github_pages)
  • category: Content category
  • language: Language of the nasheed
  • tags: Searchable tags
